Structural Terminal

This is my attempt at a structural design "terminal". When I was designing wood framed buildings in Boston there were maybe 10 resources I wanted to be able to access from one place as quickly as possible.

Godel terminal for screening stocks, gave me the idea to set up the beta version of this product. One of these days I will take another pass at it. For now, type in LVL and click enter to see some table values, without searching through folders.

Grids

Add the grid class to a container to divide up the horizontal space evenly for the cells. Note that it maintains the monospace, so the total width might not be 100%. Here are six grids with increasing cell count:

If we want one cell to fill the remainder, we set flex-grow: 1; for that particular cell.
